- Smooth animations and transitions

## How to Use

1. Open `index.html` in your web browser
2. Click the buttons or use your keyboard to perform calculations

### Keyboard Shortcuts

- `0-9`: Number input
- `+, -, *, /`: Operators
- `Enter` or `=`: Calculate result
- `Escape`: Clear all
- `Backspace`: Delete last digit
- `.`: Decimal point
- `%`: Percentage

## Files

- `index.html`: Main HTML structure
- `styles.css`: Modern CSS styling with glassmorphic effects
- `script.js`: Calculator logic and functionality

## Technologies Used

- HTML5
- CSS3 (with gradients, backdrop-filter, and animations)
- Vanilla JavaScript (ES6+ with classes)
